Answering Some Important Question

1.What is DOM and what is the purpose of DOM?
Ans: DOM stands for Document Object Model. It is a programming API for HTML and XML documents. It's used to access the content, structure, and style of a document. The DOM is what allows you to change the content and design on structure via JavaScript.


2.How will you select HTML elements using DOM? Name the DOM methods.
Ans: There are 5 ways:
  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()


3.What is a callback function and why will you use it?
Ans: When you want one function to execute only after another function has completed its execution, we use callback functions in JavaScript. It needs to pass as a parameter to other functions to make a function callback.


4. What is event bubble?
Ans: Event Bubbling is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.
